### 2. Create

POST api/v1/nodes/

Create an node with the given parameters.

Body ParametersTypeNotenamestring  
typestring  
numint

Example Requst Body

```
{
    "name": "orderer",
    "type": "orderer",
    "num": 1
}
```

Example Response Body

```
{
    "data": {
        "id": "efdcbc5b-2695-40fa-93ea-c20bc5eabdcd"
    },
    "msg": null,
    "status": "successful"
}

```
