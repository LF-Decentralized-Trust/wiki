#### Code &amp; Projects

The [Global Blockchain Business Council has just published a report about Learning Tokens](https://www.gbbc.io/news/gbbc-and-hyperledger-foundation-release-learning-tokens-special-report).  The report that dives into blockchain’s role in representing the value of transmission of knowledge by tokenizing the learning process. The Learning Tokens system is built using Hyperledger Besu and this comes out of work down as part of Hyperledger's Annual Mentorship program. Work around Learning Tokens is happening in Hyperledger Labs and you can [check out the Learning Tokens repo](https://github.com/hyperledger-labs/learning-tokens) to learn more and get involved.
