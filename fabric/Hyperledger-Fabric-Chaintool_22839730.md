# Hyperledger Fabric : Hyperledger Fabric Chaintool

Created by Tracy Kuhrt, last modified by Ry Jones on Sep 25, 2019

Chaintool is tooling to assist in various phases of Hyperledger FabricÂ chaincode development, such as compilation, test, packaging, and deployment. A chaincode application developer may express the programmatic interface to their application in a high-level protobuf-like structure, and chaintool will generate:

1. the chaincode methods stub
2. a chaincode archive package so the application may be deployed

# Key Characteristics

Integrated with Fabric, written in Clojure
