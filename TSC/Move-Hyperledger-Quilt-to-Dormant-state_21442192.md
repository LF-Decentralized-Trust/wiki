# Technical Oversight Committee : Move Hyperledger Quilt to Dormant state

Created by Arnaud J LE HORS, last modified on Mar 10, 2022

  **Status**

RESOLVED 

**Blocking****Outcome**Unanimously approved**Minutes Link**[2021 06 17 TSC Meeting Record](2021-06-17-TSC-Meeting-Record_21442104.html)

# Overview of Proposal

Following up on the Quilt maintainer report that the project will be going into a dormant state for a while we have added a Dormant state to the project lifecycle and are now ready to officially move Quilt to that state.

# Formal Proposal(s)

Move Hyperledger Quilt to Dormant state.
