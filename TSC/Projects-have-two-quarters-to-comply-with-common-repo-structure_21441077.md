# Technical Oversight Committee : Projects have two quarters to comply with common repo structure

Created by Ry Jones, last modified by María Teresa nieto on Feb 12, 2021

  **Status**

RESOLVED 

**Outcome**Unanimously approved**Minutes Link**[28 JAN 2020](https://lf-hyperledger.atlassian.net/wiki/display/TSC/2021+01+28+TSC+Meeting+Record)

# Overview of Proposal

In the quarterly report for Q2 2021, each project must report repos that do or don't have repolinter.json and are or are not using non-master branch as default.

# Formal Proposal(s)

Projects must have repolinter.json in place [as decided previously](https://lf-hyperledger.atlassian.net/wiki/display/TSC/Common+Repo+structure); and move from master to main as the default branch. These points will have to be addressed as part of the quarterly reports.

# Action Items

- Add the following questions to the quarterly reports template:
  
  1. Have you switched from master to main in all your repos?
  
  2. Have you implemented repolinter.json in all your repos?
- Inform all maintainers via all possible venues
- Request answers to those questions when quarterly reports do not provide any answers
